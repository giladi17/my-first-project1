## 🐳 אפליקציה (App & Dockerization)

חלק זה בפרויקט מכיל את קוד האפליקציה. אנו אורזים את האפליקציה לתוך קונטיינר של Docker, ולאחר מכן מעלים את תמונת ה-Docker (Image) למאגר התמונות המנוהל של AWS (Elastic Container Registry - ECR).

### 🛠 דרישות מוקדמות
לפני ביצוע הפעולות, יש לוודא שמותקנים הכלים הבאים:
* **Docker:** מותקן ורץ ברקע (על מנת לבנות ולתייג את התמונה).
* **AWS CLI:** מותקן ומוגדר עם הרשאות הגישה המתאימות לחשבון.

### 🚀 בנייה ודחיפה של התמונה ל-ECR (Workflow)

כדי לארוז ולדחוף את האפליקציה למאגר, יש לפתוח את הטרמינל בתוך תיקיית האפליקציה (היכן שממוקם קובץ ה-`Dockerfile`) ולהריץ את ארבע הפקודות הבאות:

**1. התחברות ל-ECR:**
קבלת סיסמת התחברות זמנית מ-AWS והתחברות לשירות הדוקר המקומי מול המאגר בענן:
```bash
aws ecr get-login-password --region us-east-1 | docker login --username AWS --password-stdin 066380525112.dkr.ecr.us-east-1.amazonaws.com
